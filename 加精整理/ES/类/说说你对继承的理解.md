## 原型链

### 关系

每一个构造函数都有一个原型对象，通过该构造函数创建的实例对象也拥有一个指针指向原型对象。当原型对象指向“待继承”构造函数的实例时，就形成了原型链同时实现继承效果。

```
Sub  ---> Sub.prototype
|           |  __proto__
|           |
sub-----

```

### 特点

1. 可重写原型链
2. 原型链查找机制
3. 遮蔽效果
